How to Sideload iOS Apps Using Developer Certificate

By Neo & Trey • Neosign

Have a developer certificate and don't know how to use it? This guide will break down the process step by step to make sideloading easier for you!

Requirements

  • A developer certificate
  • The Esign (No Logs) .ipa file
  • While this guide is made for Esign, I have linked the Feather .ipa as well if you want to give that a shot
  • Other .ipa files you want to install

1. Installing ESign Using an Online .ipa Installer

First, we need to get ESign onto your device using an online .ipa installer. For the purposes of this tutorial, we have chosen Kravasign as it is one of the most reliable options.

Head to this link and fill out all the information it requires. You will need to provide the .ipa file, the certificate, and the password.

After uploading all your files, press upload. You should then get a direct install link, asking you to download the Esign app to your device.

2. Setting Up ESign

If this is your first time sideloading any developer apps, you may need to turn on developer mode in settings. You can do this by going to Settings > Privacy and Security > Developer Mode.

3. Importing Your Developer Certificate

To begin using ESign to sideload apps, you'll need to import your developer certificate. Here's how:

  1. In ESign, navigate to the Files tab (leftmost tab in the interface)
  2. Press on the 3 dots on the top right corner and locate the import button
  3. Select your certificate .zip file when prompted
  4. Once imported, tap the .zip file; Press unzip to extract its contents
  5. After extraction, you can safely delete the original .zip file
  6. Open the folder, and import the .p12 and .mobileprovision files

4. Signing and Installing Apps

Now that your certificate is imported, you can begin signing and installing apps:

  1. Press on the 3 dots on the upper right corner and import your .ipa file
  2. After importing, the .ipa, tap on the .ipa file and press Import App library
  3. Head over to the apps tab; This is the 2nd tab from the left in the interface
  4. Tap on the app you want to install
  5. Don't tap the "Install" button directly - this won't work!
  6. Instead, tap "SIGNATURE" first
  7. When prompted, tap "Signature" again to begin the signing process
  8. Once signing is complete, tap "Install"
  9. Approve the installation prompt that appears
  10. You should now see the signed app installed on your homescreen/app library

Tips for Successful Installation

  • Make sure your device has sufficient storage space
  • Keep your device connected to stable internet during the signing process
  • Verify that Developer Mode is properly enabled in Settings
